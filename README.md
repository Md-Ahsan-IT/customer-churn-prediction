# Customer Churn Prediction using Machine Learning and Deep Learning

## Project Overview

Customer churn is a major challenge in the telecommunications industry. This project analyzes customer behavior and predicts whether a customer is likely to leave the company.

The project combines:

* Exploratory Data Analysis (EDA)
* Machine Learning Models
* Artificial Neural Network (ANN)
* Business Recommendations using Generative AI

## Dataset

Dataset: Telco Customer Churn Dataset

Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Objectives

* Analyze customer churn patterns
* Identify key factors affecting churn
* Build predictive Machine Learning models
* Develop an Artificial Neural Network for churn prediction
* Generate business recommendations for customer retention

## Deep Learning Model

An Artificial Neural Network (ANN) was developed using TensorFlow/Keras and compared with Machine Learning models.

## Key Findings

* Customers with month-to-month contracts showed higher churn rates.
* Higher monthly charges were associated with increased churn.
* Customers with shorter tenure were more likely to leave.
* Long-term contracts significantly reduced churn probability.

## Business Recommendations

1. Introduce loyalty rewards for long-term customers.
2. Offer personalized retention campaigns for high-risk customers.
3. Promote annual and two-year contract plans.
